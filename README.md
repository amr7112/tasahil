# تساهيل مالي (Tasahil Mali)

**تطبيق تساهيل مالي** هو رفيقك الرقمي الأول لتسجيل الديون، متابعة التحصيلات، وإصدار التقارير المالية بكل أمان وسهولة.

### 🌐 الموقع الرسمي:
[اضغط هنا لزيارة موقع تساهيل مالي](https://amr7112.github.io/tasahil/)

### 💡 نبذة عن النظام:
"تساهيل مالي" هو نظام متكامل لإدارة الحسابات والديون، مصمم ليقدم تجربة مستخدم سلسة باللغة العربية، ويوفر حلولاً فعالة للأنشطة التجارية المختلفة.
